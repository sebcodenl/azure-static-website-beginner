Azure Beginner Project

Mijn eerste Azure project

Dit is mijn eerste website die ik heb gehost in Microsoft Azure. Met dit project bouw ik stap voor stap mijn cloud skills en GitHub-portfolio op.

Over mij

Ik ben bezig om Azure specialist te worden. Ik volg nu AZ-900 en wil later AZ-104 doen om te ontdekken wat ik het leukste vind binnen de cloud.

Wat ik in dit project leer

  • Werken met een Azure Resource Group
  • Een Storage Account aanmaken
  • Static Website Hosting activeren
  • Bestanden uploaden naar Azure
  • Een project documenteren in GitHub

Waarom dit project

Ik wilde beginnen met een simpel en overzichtelijk project dat goed laat zien dat ik niet alleen theorie leer, maar ook echt iets in Azure bouw.

Volgende stappen

  • Mijn website verder verbeteren
  • Deployment automatiseren met GitHub Actions
  • Meer Azure-projecten toevoegen aan mijn portfolio
